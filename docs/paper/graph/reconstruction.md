# 재구성

재구성에서 확인할 것은 3D 후보의 단면이 원래 2D 조각 이미지와 같은 패턴 수치를 갖는가다. [1](#ref-1)

여기서 정답은 실제 재료 속에 숨어 있던 하나뿐인 3D 구조가 아니다. 논문이 가진 것은 2D 조각 이미지다. 그래서 검증도 3D 후보를 잘랐을 때, 그 단면들이 원래 2D 데이터와 얼마나 맞는지로 한다. [2](#ref-2) [3](#ref-3)

패턴 수치로 쓴 것은 [[docs/paper/graph/s2|S2]]다. S2는 덩어리 하나의 크기를 직접 재는 수치가 아니라, 같은 물질이 거리별로 얼마나 같이 나타나는지 보는 수치다. 그래서 이 논문은 그림만 비교하지 않고, 3D 후보를 잘라 낸 단면의 S2 곡선도 비교한다. [1](#ref-1)

S2 맞춤은 만든 3D 후보의 S2 곡선을 원래 2D 데이터 쪽으로 다시 맞추는 단계다. 논문에서는 이를 S2-guided SDS라고 부른다. [2](#ref-2) [3](#ref-3)

```text
가진 것: 2D 조각 이미지
만드는 것: 3D 후보
확인할 것: 3D 후보를 잘라 낸 단면의 S2 곡선이 원래 2D 데이터와 맞는가
```

상대 오차는 두 S2 곡선 아래 면적이 얼마나 다른지 보는 숫자다. 0%에 가까울수록 원래 2D 데이터와 만든 3D 후보의 단면 패턴이 더 가깝다. 이 숫자는 모든 3D 연결을 증명하지는 않는다. 다만 “원래 2D 조각 이미지의 거리별 패턴을 유지했는가”를 보는 검증 기준이다. [1](#ref-1)

## 두 물질 재구성

Figure 3은 두 물질로 된 재료를 3D 후보로 만든 결과다. 논문에서는 이를 binary microstructure라고 부른다. [2](#ref-2)

![Figure 3](../md/2508.20138v4-assets/figure-03-page-12.png)

- a. 원래 2D 조각 이미지다.
- b. MicroLad가 만든 3D 후보다.
- c. b를 임의 방향에서 잘라 본 단면이다.
- d. c의 S2 곡선과 원래 2D 데이터의 S2 곡선을 비교한다.
- e. S2 곡선에 더 맞도록 다시 고친 3D 후보다.
- f. e를 잘라 본 단면이다.
- g. 다시 고친 뒤의 S2 곡선 비교다.

핵심은 c와 f다. 3D 후보 자체를 눈으로만 보는 것이 아니라, 그 후보를 다시 2D 단면으로 잘라 원래 2D 조각 이미지와 같은 패턴을 보이는지 확인한다.

두 물질 재구성에서 가장 큰 S2 상대 오차는 S2 맞춤 전 4.62%, 맞춤 후 0.41%다. [4](#ref-4)

## 세 물질 재구성

Figure 4는 세 물질로 된 고체산화물 연료전지(SOFC) 재료를 3D 후보로 만든 결과다. 어두운 부분은 빈 공간, 중간 회색은 YSZ, 밝은 부분은 Ni다. [3](#ref-3)

![Figure 4](../md/2508.20138v4-assets/figure-04-page-13.png)

- a. 원래 2D 조각 이미지다.
- b. MicroLad가 만든 3D 후보다.
- c. b를 잘라 본 단면이다.
- d. c의 S2 곡선과 원래 2D 데이터의 S2 곡선을 비교한다.
- e. S2 곡선에 더 맞도록 다시 고친 3D 후보다.
- f. e를 잘라 본 단면이다.
- g. 다시 고친 뒤의 S2 곡선 비교다.

세 물질 재구성은 두 물질보다 어렵다. 물질 종류가 늘면 각 물질이 차지하는 양이 줄고, S2 값 자체도 작아진다. 논문은 빈 공간의 부피 비율이 약 0.2라서 상대 오차가 더 민감하게 커진다고 설명한다. [3](#ref-3)

세 물질 재구성에서 가장 큰 S2 상대 오차는 S2 맞춤 전 10.81%, 맞춤 후 6.13%다. [4](#ref-4)

| 경우 | S2 맞춤 전 최대 오차 | S2 맞춤 후 최대 오차 |
| --- | ---: | ---: |
| 두 물질 | 4.62% | 0.41% |
| 세 물질 | 10.81% | 6.13% |

## 비교 기준

세 물질 SOFC 비교에서 MicroLad의 최대 S2 상대 오차는 6.13%다. MPDD는 12.44%, SliceGAN은 11.01%다. [5](#ref-5)

| 방법 | 최대 S2 상대 오차 |
| --- | ---: |
| MicroLad + S2 맞춤 | 6.13% |
| [[docs/paper/graph/mpdd|MPDD]] | 12.44% |
| SliceGAN | 11.01% |

따라서 이 노드의 결론은 “실제 내부 3D 정답을 찾았다”가 아니다. 더 좁게 말하면, MicroLad는 2D 조각 이미지의 패턴 수치와 맞는 3D 후보를 만들었다.

이 확인이 끝나야 다음 질문으로 넘어갈 수 있다. 다음 질문은 목표 수치를 넣었을 때 3D 후보가 원하는 방향으로 움직이는가다. 그 내용이 [[docs/paper/graph/control|역방향 제어]]다.
