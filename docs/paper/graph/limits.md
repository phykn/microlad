# 한계

이 논문은 2D 조각 이미지에서 패턴이 맞는 3D 후보를 만들고, 필요하면 목표 수치 쪽으로 후속 조정한다. 한계는 그 후보가 실제 내부 3D 정답이거나, 모든 방향의 연결과 모든 목표를 보장한다는 뜻은 아니라는 점이다.

## 정답 하나를 찾는 것이 아니다

MicroLad가 만드는 것은 실제 재료 속에 숨어 있던 하나뿐인 3D 정답이 아니다. 2D 조각 이미지와 패턴 수치가 맞는 3D 후보다.

특히 [[docs/paper/graph/s2|S2]] 같은 패턴 수치는 구조를 일부만 본다. 논문도 두 점 상관 같은 지표만 쓰면 서로 다른 구조가 거의 같은 통계를 가질 수 있다고 말한다. 그래서 패턴 수치가 맞는 후보가 하나만 있는 것이 아니다. [1](#ref-1)

```text
맞게 읽기:
2D 조각 이미지와 패턴 수치가 맞는 3D 후보를 만든다

과하게 읽기:
실제 내부 3D 구조를 정확히 복원했다
```

## 슬라이스 가정

이 논문은 3D 후보를 세 방향 슬라이스로 다룬다. 앞-뒤, 좌-우, 위-아래 방향에서 슬라이스를 보고, 그 슬라이스들이 서로 가까운 패턴을 유지하도록 만든다. 이 가정이 [[docs/paper/graph/mpdd|MPDD]]와 [[docs/paper/graph/control|역방향 제어]]의 전제다. [2](#ref-2) [3](#ref-3)

논문은 Figure 10에서 만들어진 3D 후보의 연속 슬라이스들이 원래 독립 2D 조각 이미지보다 더 가까이 모인다는 점을 확인한다. 이 결과는 만든 3D 후보의 이웃 슬라이스들이 서로 독립적으로 흩어지지 않는다는 근거다. [4](#ref-4)

![Figure 10](../md/2508.20138v4-assets/figure-10-page-20.png)

하지만 논문은 대각선 같은 비직교 방향의 일관성은 명시적으로 보장되지 않는다고 말한다. 즉 세 방향에서 잘 맞는다는 것이 모든 방향에서 완전히 맞는다는 뜻은 아니다. [4](#ref-4)

```text
맞게 읽기:
세 방향 슬라이스 기준으로 3D 후보의 일관성을 맞춘다

과하게 읽기:
3D 공간의 모든 방향 연결이 보장된다
```

## 역방향 제어의 가정

[[docs/paper/graph/control|역방향 제어]]는 3D 후보 전체를 한 번에 고치는 방식이 아니다. 3D 후보 안의 2D 슬라이스를 하나씩 고르고, 그 슬라이스를 목표 수치 쪽으로 조금씩 고친다. 논문은 이렇게 고친 슬라이스들이 공간적으로 연결되어 있으므로, 반복하면 전체 3D 후보가 점차 목표 쪽으로 움직인다고 가정한다. [2](#ref-2)

이 가정은 읽을 때 중요하다. 슬라이스 하나가 좋아졌다고 해서 자동으로 3D 전체가 완벽해지는 것은 아니다. 여러 방향과 여러 위치의 슬라이스를 반복해서 고치기 때문에 전체 후보가 움직인다고 보는 것이다.

```text
맞게 읽기:
연결된 슬라이스를 반복 조정해서 3D 후보 전체를 목표 쪽으로 민다

과하게 읽기:
슬라이스 하나만 맞추면 3D 전체 목표가 보장된다
```

## 목표 수치의 범위

역방향 제어에서 쓰는 목표 수치도 무제한이 아니다. 논문 본문 실험은 부피 비율, 표면적, 확산도를 목표로 두고, 각 목표값을 원래 2D 학습 데이터에서 관측된 최대값으로 설정한다. 각 경우 3D 후보 50개를 만들어 분포가 목표 쪽으로 이동하는지 본다. [5](#ref-5)

이 말은 “아무 목표를 넣어도 된다”가 아니다. 논문이 보여준 것은 정해진 목표 수치에서 후보 분포가 목표 쪽으로 움직였다는 것이다.

제조 제약도 아직 중심 실험은 아니다. 논문은 입자 크기, 종횡비, 상 연결성 같은 제조 가능성 제약을 다루는 방향을 향후 과제로 제안한다. [6](#ref-6)

## 비용

3D 후보를 처음 만드는 L-MPDD는 빠른 편이다. 하지만 목표 수치로 후속 조정하는 SDS는 목표와 단계 수에 따라 비용이 커진다. 특히 확산도 목표는 FEM 계산이 들어가서 더 느리다. [7](#ref-7) [8](#ref-8)

| 작업 | 시간 |
| --- | ---: |
| 64 x 64 x 64 L-MPDD | <10초 |
| 기존 MPDD | >30분 |
| 64 x 64 x 64 부피/표면적, 500 SDS steps | ~1분 |
| 64 x 64 x 64 확산도, 500 SDS steps | ~2.5분 |
| 256 x 256 x 256 재구성 | 71초 |
| 256 x 256 x 256 확산도 후속 조정 | 156.1분 |

그래서 이 논문은 3D 촬영보다 가벼운 출발점을 제안하지만, 모든 목표 조정이 항상 가볍다는 뜻은 아니다. 특히 확산도처럼 물리 계산이 붙는 목표는 시간이 커진다.

## 다른 방식이 더 나을 때

이 논문은 목표 수치를 직접 넣는 조건부 모델을 새로 훈련하지 않는다. 대신 2D 모델과 미분 가능한 목표 계산을 이용해 SDS로 후속 조정한다. 그래서 구조와 성능이 짝지어진 라벨 데이터가 없어도 시작할 수 있다. [9](#ref-9)

하지만 논문은 데이터가 충분하고 빠른 생성 시간이 더 중요하면, 목표를 직접 입력받는 조건부 diffusion 모델이 더 적합할 수 있다고 말한다. 그런 방식은 반복 SDS를 줄일 수 있지만, 생성 가능한 범위가 학습 데이터의 조건부 분포에 묶일 수 있다. [9](#ref-9)

## 이 논문의 주장 범위

이 논문의 주장은 아래 범위로 읽는 것이 안전하다.

```text
2D 조각 이미지에서 시작한다
패턴 수치가 맞는 3D 후보를 만든다
목표가 있으면 후보를 후속 조정한다
세 방향 슬라이스와 목표 계산이 맞는 범위에서 결과를 해석한다
```

이 범위를 넘어서 “실제 3D 정답을 찾았다”, “모든 방향 연결이 보장된다”, “모든 목표를 빠르게 만족한다”로 읽으면 논문이 말한 것보다 커진다.
