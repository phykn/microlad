# S2

S2는 덩어리 하나의 크기를 직접 재는 지표가 아니라, 같은 물질이 거리별로 얼마나 같이 나타나는지 보는 패턴 수치다. 논문에서는 two-point correlation function이라고 부른다. [1](#ref-1)

## 무엇을 세는가

S2는 점 두 개를 고른다. 두 점 사이의 거리는 `r`로 정한다. 그런 다음 두 점이 같은 물질에 속하는지 본다. [1](#ref-1)

```text
1. 한 물질을 고른다
2. 점 A를 고른다
3. 점 A에서 거리 r만큼 떨어진 점 B를 고른다
4. A와 B가 둘 다 같은 물질인지 본다
5. 이 일을 많이 반복한다
6. 같은 물질일 확률을 계산한다
```

거리 `r`을 바꿔 가며 계산하면 S2 곡선이 된다.

```text
거리 1: 같은 물질일 확률
거리 2: 같은 물질일 확률
거리 3: 같은 물질일 확률
...
```

그래서 S2는 “이 구멍 하나의 지름이 몇 픽셀인가”를 말하지 않는다. 대신 “이 물질은 가까운 거리에서 자주 같이 나오나, 더 먼 거리에서도 같이 나오나”를 말한다.

## 덩어리 크기와의 관계

덩어리 크기는 S2에 흔적으로 남는다. 큰 덩어리 안에서 점 하나를 고르면, 조금 떨어진 점도 같은 덩어리 안에 있을 가능성이 높다.

```text
큰 덩어리:
가까운 거리 -> 같은 물질일 가능성 높음
조금 먼 거리 -> 여전히 같은 물질일 가능성 있음
```

작은 조각들이 흩어져 있으면 다르다. 점 하나를 고른 뒤 조금만 멀리 가도 다른 물질을 만날 가능성이 커진다.

```text
작은 조각:
가까운 거리 -> 같은 물질일 수 있음
조금 먼 거리 -> 다른 물질일 가능성 커짐
```

따라서 S2는 덩어리 크기를 간접적으로 반영한다. 하지만 덩어리 하나의 크기, 개수, 정확한 모양을 직접 알려주지는 않는다.

```text
S2로 알기 쉬운 것:
같은 물질이 거리별로 같이 나타나는 경향

S2만으로 알기 어려운 것:
덩어리 하나의 정확한 크기
덩어리 개수
두 점 사이의 실제 연결 경로
```

## 연결성과의 관계

S2는 연결성을 직접 보지 않는다. 두 점이 같은 물질인지 보지만, 두 점 사이에 실제 길이 이어져 있는지는 보지 않는다.

```text
A 지점: 빈 공간
B 지점: 빈 공간
A와 B 사이: 고체로 막힘
```

이 경우에도 S2 입장에서는 A와 B가 둘 다 빈 공간이다. 그래서 같은 물질로 센다. 중간이 막혔는지는 S2 계산에 직접 들어가지 않는다.

그래도 연결된 구조의 흔적은 S2에 간접적으로 남는다. 빈 공간이 길게 이어져 있으면, 특정 거리에서 빈 공간 두 점이 같이 잡힐 확률이 높아진다. 반대로 빈 공간이 잘게 끊겨 있으면, 조금 떨어진 두 점이 모두 빈 공간일 확률이 낮아진다.

하지만 결론은 좁게 잡아야 한다.

```text
S2가 맞다
= 거리별 물질 배치 패턴이 비슷하다

S2가 맞다
!= 실제 연결 경로가 모두 같다
```

연결 자체를 보려면 연결성, 확산도, 길이 얼마나 돌아가는지를 보는 기준이 필요하다. 이 논문에서도 S2는 재구성 패턴 검증에 쓰이고, 목표 수치 조정에서는 부피 비율, 표면적, 확산도 같은 수치가 따로 등장한다. [2](#ref-2)

## 이 논문에서의 역할

이 논문은 3D 후보가 원래 2D 조각 이미지와 비슷한지 확인할 때 S2를 쓴다. 절차는 단순하다.

```text
원래 2D 조각 이미지의 S2 곡선을 구한다
3D 후보를 슬라이스한다
그 슬라이스의 S2 곡선을 구한다
두 곡선이 얼마나 다른지 본다
```

상대 오차는 두 S2 곡선 아래 면적이 얼마나 다른지 보는 숫자다. 0%에 가까울수록 두 곡선이 가깝다. [1](#ref-1)

그래서 [[docs/paper/graph/reconstruction|재구성]] 노드에서 S2는 “3D 후보가 실제 내부 정답인가”를 증명하는 수치가 아니다. 더 정확히는 “3D 후보를 잘라 낸 단면이 원래 2D 조각 이미지의 거리별 패턴과 맞는가”를 확인하는 수치다.
